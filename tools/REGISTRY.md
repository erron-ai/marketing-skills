## By Category

### Analytics

Track user behavior, measure conversions, and evaluate marketing channel performance.

| Tool | Best For | MCP Available |
|------|----------|:-------------:|
| **ga4** | Web analytics, Google Ads attribution | ✓ |
| **mixpanel** | Product analytics, funnel analysis | - |
| **amplitude** | Behavioral analytics, cohort analysis | - |
| **posthog** | Self-hosted analytics, session replay, feature flags | - |
| **segment** | Customer data platform, event routing | - |
| **adobe-analytics** | Enterprise web analytics | - |
| **plausible** | Privacy-first, cookieless analytics | - |

**Recommendation**: Start with GA4 if you're in the Google ecosystem. Add Mixpanel or Amplitude for deeper product analytics. Use Segment to route events to multiple downstream tools. Plausible for privacy-focused teams.

### SEO

Keyword research, rank tracking, backlink analysis, and site audit tools.

| Tool | Best For | Notes |
|------|----------|-------|
| **google-search-console** | Free, authoritative search performance data | Direct from Google |
| **semrush** | Competitive keyword research, rank tracking | All-in-one |
| **ahrefs** | Backlink analysis, content gap research | Best for link data |
| **dataforseo** | Programmatic SERP and backlink data | API-first |
| **keywords-everywhere** | Browser-based keyword research | Credit-based |

**Recommendation**: Google Search Console is essential — it's free and authoritative. Layer in Semrush or Ahrefs for competitive research. Use DataForSEO for programmatic access to SERP data at scale.

## CLI Tools

Zero-dependency, single-file Node.js CLIs for tools that don't ship their own. See [`clis/README.md`](clis/README.md) for setup and usage.

All CLIs follow the same conventions:
- **No dependencies** — requires Node 18+ only, uses native `fetch`
- **JSON output** — pipe to `jq`, save to file, or use in scripts
- **Env var auth** — set `{TOOL}_API_KEY` before running
- **Consistent structure** — `{tool} <resource> <action> [--options]`
