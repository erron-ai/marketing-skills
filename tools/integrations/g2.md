# G2

B2B software review platform — the primary source of social proof for SaaS buyers. Essential for category visibility and peer recommendations.

## Capabilities

| Integration | Available | Notes |
|-------------|-----------|-------|
| API | ✓ | G2 API (select partners) |
| MCP | - | Not available |
| CLI | ✓ | See clis/g2.js |
| SDK | - | Use API directly |

## Authentication

- **API Key**: Available to G2 partners — request access via G2 account
- Pass as `X-Api-Key: {api_key}` header

## Common Agent Operations

### Get reviews for your product

```bash
GET https://data.g2.com/api/v1/products/{product_id}/reviews
X-Api-Key: {api_key}
```

### Get review summary stats

```bash
GET https://data.g2.com/api/v1/products/{product_id}
X-Api-Key: {api_key}
```

Returns: overall rating, total reviews, rating distribution, category position.

## Review Generation Strategy

G2 reviews are critical for:
- Appearing in G2 category lists and Grid reports
- Buyer research — most SaaS buyers check G2 before evaluating
- Social proof on your website (G2 badges and widgets)

**How to generate reviews:**
1. Send personalized email invitations to your most successful customers
2. Offer a gift card (G2 Gift Card program — compliant with G2 guidelines)
3. Time asks after success moments (renewal, expansion, key milestone)
4. Respond to all reviews — positive and negative

## G2 Categories and Badges

Maintaining a strong G2 presence means:
- Claiming and filling out your G2 profile completely
- Reaching minimum review thresholds for category Grid placement
- Earning Leader, High Performer, or Momentum badges for use in marketing
