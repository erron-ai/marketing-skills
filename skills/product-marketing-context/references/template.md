# Product Marketing Context Template

A fill-in-the-blank template for the context document. Copy this structure when creating `.agents/product-marketing-context.md`.

## Sections

See the main SKILL.md for a complete description of each section and what to capture.

### Minimal Viable Context

If you're in a hurry, fill in at minimum:
1. Product Overview (one-liner + what it does)
2. Target Audience (who it's for)
3. Problems & Pain Points (what problem it solves)
4. Differentiation (why choose you over alternatives)

Other skills can work with just these four sections. Expand the document over time as you use more skills.
