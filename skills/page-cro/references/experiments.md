# Page CRO Experiment Ideas

A reference library of testable hypotheses organized by page type.

## Homepage Experiments

### Hero Section
| Test | Hypothesis |
|------|------------|
| Headline variants | Outcome-focused vs. feature-focused messaging |
| Subheadline support | Clarify or remove the subheadline |
| Above-fold CTA | Test presence, placement, and wording |
| Hero visual format | Product screenshot vs. explainer GIF vs. illustration |
| CTA button text | "Start Free Trial" vs. "Get Started" vs. "See It in Action" |
| Interactive product preview | Let visitors explore the product before signing up |

### Trust and Social Proof
| Test | Hypothesis |
|------|------------|
| Logo placement | In the hero vs. below the fold |
| Testimonial in hero | Lead with a strong customer quote |
| Review badge | Show G2 or similar rating in the header |
| Social proof in headline | Incorporate "Join X,000+ teams" into the headline |

### Content and Features
| Test | Hypothesis |
|------|------------|
| Feature order | Lead with the feature users value most |
| Benefits vs. features | Reframe feature descriptions as outcomes |
| Comparison section | Show how you compare to the status quo |
| Secondary CTAs | Add or remove CTAs mid-page |

### Navigation and UX
| Test | Hypothesis |
|------|------------|
| Sticky nav with CTA | Persistent header with a sign-up button |
| Nav item order | Most important items at edges of the nav |
| Exit intent capture | Show offer or prompt when visitor moves to leave |
| Live chat vs. bot | Impact of real-time help on conversion |

## Pricing Page Experiments

### Plan Presentation
| Test | Hypothesis |
|------|------------|
| Recommended plan badge | "Most popular" or "Best for teams" label |
| Monthly vs. annual default | Show annual pricing first to anchor value |
| Annual toggle prominence | Make savings visible before visitor has to look |
| Plan naming | Descriptive names vs. abstract tiers (Starter/Growth/Scale) |

### Objection Handling
| Test | Hypothesis |
|------|------------|
| Money-back guarantee | Reduce perceived risk near the CTA |
| FAQ below pricing | Address the top pricing objections in place |
| Trial length emphasis | "No credit card required" vs. "14-day free trial" |

## Landing Page Experiments

### Message Match
| Test | Hypothesis |
|------|------------|
| Headline mirrors ad copy | Match language exactly from the ad that drove the click |
| Source-specific landing pages | Unique page for each major traffic source |

### Conversion Focus
| Test | Hypothesis |
|------|------------|
| Remove navigation | Landing page without a nav vs. with a nav |
| Single CTA vs. two options | One action vs. "Start Trial" + "Book Demo" |
| Form position | Form above the fold vs. after the argument is made |

## Feature Page Experiments

### Feature Presentation
| Test | Hypothesis |
|------|------------|
| Lead with outcome | Outcome-first headline vs. feature name first |
| Before/after visual | Show the state before and after using the feature |
| Use case specificity | Generic use case vs. industry-specific scenario |

### Conversion Path
| Test | Hypothesis |
|------|------------|
| Inline trial CTA | CTA within the feature description |
| Demo video | Short product demo vs. static screenshot |

## Blog and Content Page Experiments

### CTA Format
| Test | Hypothesis |
|------|------------|
| Inline text CTA | Contextual link within the body copy |
| Banner card | Styled CTA block after a key section |
| Sticky sidebar CTA | Persistent CTA as reader scrolls |
| Exit intent for content | Offer a related asset as reader exits |

### CTA Relevance
| Test | Hypothesis |
|------|------------|
| Topic-matched CTA | CTA directly related to the post topic vs. generic signup |
| Content upgrade | Offer a downloadable version of the post content |
